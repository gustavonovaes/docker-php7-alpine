# docker-php7-nginx-alpine

## Versions

```
Nginx: 1.15
PHP  : 7.2
```

## Run

```sh
$ git clone https://github.com/gustavonovaes/docker-php7-nginx-alpine.git
$ cd docker-php7-nginx-alpine
$ docker-compose up 
```

## Log

The logs are stored in two files `nginx.log` and `php-fpm.log` on directory `/var/log/`.
